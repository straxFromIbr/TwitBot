# TwitBot with markov-chain

## dependencies
- bash
- twurl : 0.9.6
- Python3 : 3.8.10
    - mecab-python3 : 1.0.4
        - 辞書：[neologd/mecab-ipadic-neologd](https://github.com/neologd/mecab-ipadic-neologd)
    - markovify : 0.9.3

## Execution
2 . 必要ライブラリのインストール。`pip`と`gem`
1. `twurl`の認証. 要Twitter API
2. `twitbot.sh`ファイル中のパスの設定
3. `bash twitbot.sh`
